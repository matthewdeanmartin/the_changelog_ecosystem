`scriv` is a python cli tool for managing changelogs and releases.

Changelog management CLI that creates and aggregates changelog fragments into a project changelog.

## Installation

```bash
pip install scriv
# or with uv:
uv add scriv
```

## What It Does

- Assembles changelog from individual news/change fragment files
- Writes and updates a `CHANGELOG.md` file
- Edit
- Collect
- Pyproject config
