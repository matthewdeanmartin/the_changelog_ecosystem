`reno` is a python cli sphinx tool for managing changelogs and releases.

OpenStack release-notes manager that stores note files and associates them with releases based on git tags.

## Installation

```bash
pip install reno
# or with uv:
uv add reno
```

## What It Does

- Assembles changelog from individual news/change fragment files
- Generates release notes for GitHub/GitLab releases
- Sphinx
- Git tags
- Branch aware
