`auto-changelog` is a npm cli tool for managing changelogs and releases.

Command-line tool that generates changelogs from git tags and commit history.

## Installation

```bash
npm install --save-dev auto-changelog
# or globally:
npm install -g auto-changelog
```

## What It Does

- Generates changelog from git tags
- Generates changelog/release notes from git commit history
- Templates
- Can generate changelog from existing history / backfill old releases
