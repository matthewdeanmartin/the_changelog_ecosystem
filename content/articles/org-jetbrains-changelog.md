`org.jetbrains.changelog` is a gradle plugin tool for managing changelogs and releases.

JetBrains Gradle plugin with tasks and helpers for working with Keep a Changelog style changelogs.

## Installation

Add to `build.gradle.kts`:
```kotlin
plugins {
    id("org.jetbrains.changelog")
}
```

## What It Does

- Implements or targets the Keep a Changelog format
- Gradle task
- Patch changelog
- Generates release notes for GitHub/GitLab releases
- Intellij plugin
