`changelogithub` is a npm cli tool for managing changelogs and releases.

CLI for generating GitHub release changelogs from Conventional Commits, powered by changelogen.

## Installation

```bash
npm install --save-dev changelogithub
# or globally:
npm install -g changelogithub
```

## What It Does

- Creates or updates GitHub Releases
- Parses Conventional Commits message format
- Generates release notes for GitHub/GitLab releases
- Contributors
- Update existing release
