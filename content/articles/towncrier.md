`towncrier` is a python cli tool for managing changelogs and releases.

Building newsfiles for your project via news fragments

## Installation

```bash
pip install towncrier
# or with uv:
uv add towncrier
```

## What It Does

- Assembles changelog from individual news/change fragment files
- Writes and updates a `CHANGELOG.md` file
- Generates release notes for GitHub/GitLab releases
- Issue links
- Markdown
