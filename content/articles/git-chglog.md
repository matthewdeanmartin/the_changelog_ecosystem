`git-chglog` is a go cli tool for managing changelogs and releases.

Go-based changelog generator that uses git tags and commits with configurable templates.

## What It Does

- Generates changelog/release notes from git commit history
- Generates changelog from git tags
- Templates
- Can generate changelog from existing history / backfill old releases
- Parses Conventional Commits message format
