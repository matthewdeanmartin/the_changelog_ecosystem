`git-cliff` is a cli tool for managing changelogs and releases.

Highly customizable changelog generator from git history

## Installation

```bash
cargo install git-cliff
```

## What It Does

- Generates changelog/release notes from git commit history
- Parses Conventional Commits message format
- Supports custom output templates
- Can generate changelog from existing history / backfill old releases
- Generates release notes for GitHub/GitLab releases
