## On the Radar (13)
- [@changesets/changelog-github](https://github.com/changesets/changesets) — Changesets changelog generator that links changelog entries to GitHub commits, PRs, and users.
- [@semantic-release/release-notes-generator](https://github.com/semantic-release/release-notes-generator) — semantic-release plugin that generates release note content with conventional-changelog.
- [auto-changelog](https://github.com/CookPete/auto-changelog) — Command-line tool that generates changelogs from git tags and commit history.
- [changelogithub](https://github.com/antfu/changelogithub) — CLI for generating GitHub release changelogs from Conventional Commits, powered by changelogen.
- [Changesets](https://github.com/changesets/changesets#main) — File-based release intent workflow for packages and monorepos; contributors add changeset files that drive version bumps
- [commit-and-tag-version](https://github.com/absolute-version/commit-and-tag-version) — Drop-in replacement style tool for npm version workflows with automated bumping, tagging, and changelog generation.
- [conventional-changelog](https://github.com/conventional-changelog/conventional-changelog) — Core Node ecosystem changelog toolkit for generating changelogs and release notes from commit metadata.
- [conventional-changelog-cli](https://github.com/conventional-changelog/conventional-changelog) — Generate changelogs from conventional commits
- [github-release-from-changelog](https://github.com/MoOx/github-release-from-changelog) — CLI that extracts the relevant changelog section and creates/updates a GitHub Release for a tag.
- [keep-a-changelog](https://github.com/oscarotero/keep-a-changelog) — Node/Deno library for parsing and generating Keep a Changelog formatted changelog files.
- [release-it](https://github.com/release-it/release-it) — Generic CLI tool to automate versioning and package publishing
- [semantic-release](https://github.com/semantic-release/semantic-release) — Fully automated version management and package publishing
- [standard-version](https://github.com/conventional-changelog/standard-version) — Automate versioning and CHANGELOG generation with semver and conventional commits
