## Reviewed (6)
- **[dotnet-releaser](/reviews/dotnet-releaser/)** — All-in-one .NET release CLI for building, testing, packaging, creating release notes from PRs/commits, publishing to NuGet, and creating GitHub releases.
- **[GitReleaseManager](/reviews/gitreleasemanager/)** — .NET/GitTools utility for creating draft releases from milestones, attaching assets, publishing drafts, and exporting release notes.
- **[GitReleaseNotes](/reviews/gitreleasenotes/)** — GitTools utility for generating release notes from GitHub, Jira, and YouTrack project data.
- **[GitVersion](/reviews/gitversion/)** — Git-based semantic version calculator for .NET and CI pipelines.
- **[Nerdbank.GitVersioning](/reviews/nerdbank-gitversioning/)** — Microsoft/dotnet project for stamping assemblies, NuGet packages, VSIX, and other artifacts with Git-derived SemVer information.
- **[versionize](/reviews/versionize/)** — .NET tool for automatic NuGet versioning and CHANGELOG generation from Conventional Commits.
