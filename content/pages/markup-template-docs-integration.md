## Purpose

Generic documentation tooling is too large for this site. This section is deliberately narrower: the markup formats and template systems changelog tools actually prefer, plus the integration points needed to get release notes into docs sites and release pages.

## Scope Limits

- In scope: Markdown, reStructuredText, Handlebars, Tera, Go templates, Jinja-style templates, release-body templates, and docs-site handoff.
- Out of scope: general static site generation, full documentation portals, API reference generation, and prose authoring systems unrelated to release notes.
