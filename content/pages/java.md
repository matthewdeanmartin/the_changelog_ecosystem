## On the Radar (5)
- [com.diffplug.spotless-changelog](https://github.com/diffplug/spotless-changelog) — Gradle plugin that checks Keep a Changelog compliance and can compute release versions from the changelog.
- [git-changelog-maven-plugin](https://github.com/tomasbjerre/git-changelog-maven-plugin) — Maven plugin that generates changelogs or release notes from a git repository, with issue tracker integration.
- maven-changelog-plugin — Apache Maven plugin that generates SCM change reports for Maven sites, including changelog and activity reports.
- [net.wooga.github-release-notes](https://github.com/wooga/atlas-release) — Gradle plugin providing tasks and conventions for GitHub release-note messages.
- [org.jetbrains.changelog](https://github.com/JetBrains/gradle-changelog-plugin) — JetBrains Gradle plugin with tasks and helpers for working with Keep a Changelog style changelogs.
