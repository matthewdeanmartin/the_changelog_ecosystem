I forked keepachangelog-manager (now named keepachangelog-manager-fork) and realized it is undiscoverable.

If happen to be looking for a changelog/release manager tool consider it if

- You like keepachangelog 1.1.0 as your format standard
- You use semantic version, pep 440, or calver as your version standard
- You don't already have another preferred version bumper
- You prefer Markdown for your templating language
- You use Github or Gitlab as your Release page.

Otherwise, browse this site and pick one of the other tools.

I create this site with a mix of Sonnet, Codex, etc. This is slopumentation, but with quality checks.
